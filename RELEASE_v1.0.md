## Core product definition

```text
ESPAndyAI EDGE is a physical edge-node product line for ESP32-class devices connected to AndyAI Director.

AndyAI Director provides memory, governance, policy, orchestration, approval, and dashboard control.

ESPAndyAI EDGE provides microphone, speaker, local logs, sensors, buttons, Wi‑Fi communication, and physical presence.
```
