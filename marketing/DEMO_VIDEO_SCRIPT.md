# Demo Video Script — 90 Seconds

## Title

**ESPAndyAI EDGE — Give Your AI a Body**

## Scene 1 — Opening

Show small device on desk.

Voiceover:

```text
Most AI tools live inside a screen.
But real work happens in rooms, offices, workshops, and physical spaces.
```

## Scene 2 — Problem

Show laptop, documents, messy notes, workshop tools.

Voiceover:

```text
IoT devices can sense and switch.
Chatbots can talk.
But businesses need something more: a physical AI node that listens, logs, answers, and acts under human control.
```

## Scene 3 — Device Interaction

Andy presses button.

Andy says:

```text
AndyAI, save this as a new idea for Director Edge Node.
```

Device responds:

```text
Saved. The event was sent to Director and logged locally.
```

## Scene 4 — Dashboard

Show Director dashboard event.

Voiceover:

```text
The node captures the command, sends it to AndyAI Director, receives a decision, and stores a local log.
```

## Scene 5 — Safe Action

Andy says:

```text
AndyAI, turn on demo mode.
```

Device turns LED on or shows status.

Voiceover:

```text
Safe commands can be executed immediately. Risky commands can require approval.
```

## Scene 6 — Use Cases

Show quick visual cards:

```text
Office Node
Docs Node
Workshop Node
Education Node
Custom Business Deployment
```

## Scene 7 — Closing

Voiceover:

```text
ESPAndyAI EDGE is not just a gadget.
It is a physical voice and action layer for AndyAI Director.
```

