# Hardware Specification

## v0.1 Prototype Target

### Core

```text
ESP32-S3 development board
USB-C power
Wi-Fi connection
```

### Audio

```text
I2S digital microphone
I2S amplifier
small speaker
push-to-talk button
```

### Storage

```text
microSD card module
local event log
```

### Interface

```text
1–2 buttons
OLED display optional
status LED
```

### Enclosure

```text
simple 3D printed case
desktop form factor
access to USB-C
access to reset/boot buttons during development
```

## v0.2 Engineering Prototype

Add:

```text
cleaner enclosure
stable wiring or simple carrier board
device label / QR code
better power handling
status screen
mounting points
```

## v1.0 Dev Kit

Add:

```text
repeatable assembly
setup guide

```

## Production Principles

```text
use pre-certified modules where possible
avoid custom RF in early versions
support one main board first
reduce component variants
keep enclosure simple
design for testability
```
