# Financial Model

## 1. Revenue Streams

```text
hardware sales
Director SaaS subscription
pilot deployment fees
custom integration
support and maintenance
licensing
```

## 2. Early Pricing

| Product | Suggested Price |
|---|---:|

| SaaS Business | $199–$499/month |
| Light Pilot | $1,500–$3,000 |

| Advanced Pilot | $15,000–$30,000 |
| Custom Deployment | $25,000–$250,000+ |

## 3. Scenario A — Lean Validation

```text
Budget: $5,000–$25,000
Units: 20–50
Revenue target: $5,000–$25,000
Goal: validation, demo, first pilot
```

## 4. Scenario B — Small Commercial Pilot

```text
Budget: $30,000–$100,000
Units: 100–300
Revenue target: $50,000–$200,000
Goal: paid pilots, dev kit sales, SaaS start
```

## 5. Scenario C — Investor-Backed First Product

```text
Budget: $250,000–$500,000
Units: 500–1,500
Revenue potential: $150,000–$750,000+
Goal: team, productization, certification planning, sales
```

## 6. Key Metrics

```text
hardware gross margin
monthly recurring revenue
API cost per active node
number of active nodes
number of commands per node
pilot conversion rate
waitlist conversion rate
support tickets per device
device return rate
customer retention
```

## 7. Business Logic

Hardware opens the door.  
SaaS creates recurring revenue.  
Pilots fund learning.  
Custom deployments create high-value proof.  
The platform becomes valuable as device count and workflow templates grow.
