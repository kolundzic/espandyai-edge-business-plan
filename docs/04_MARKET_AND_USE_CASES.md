# 04 — Market and Use Cases

## 1. Market Direction

The market is moving toward physical AI, AIoT, edge AI, voice agents, and local workflow automation.

The opportunity is not simply that small devices can connect to AI.

The opportunity is that small devices can become governed, voice-enabled, task-aware extensions of business workflows.

## 2. Target Market Segments

### Segment 1 — Developers and Makers

They want early access, flexibility, documentation, and hackability.

### Segment 2 — Small Businesses

They need practical outcomes: notes, tasks, office workflow, documentation access, simple automation.

### Segment 3 — Documentation-Heavy Teams

They need voice access to internal knowledge, manuals, procedures, and support material.

### Segment 4 — Workshops, Labs, Repair Shops

They need hands-free logging, measurements, status notes, and quick access to technical procedures.

### Segment 5 — Education and Publishing

They need learning companions that allow people to talk to educational materials, books, courses, and internal knowledge bases.

### Segment 6 — Integrators and AI Agencies

They can use ESPAndyAI EDGE as a physical layer for their own customer deployments.

## 3. Priority Use Cases

### Use Case A — Office Voice Node

Commands:

```text
Save this idea.
Create a task.
Read today’s project status.
Send this note to Director.
```

### Use Case B — Documentation Voice Node

Commands:

```text
What does the installation manual say?
Find the reset procedure.
Summarize the safety warning.
Show the next step.
```

### Use Case C — Workshop Node

Commands:

```text
Record measurement: 3.3 volts.
Start sensor test.
Log this repair step.
Send status to dashboard.
```

### Use Case D — Education Node

Commands:

```text
Explain this chapter.
Ask the book a question.
Read the summary.
Create a quiz from this section.
```

### Use Case E — Reception / Concierge Node

Commands:

```text
Answer visitor questions.
Give directions.
Check basic information.
Send request to staff.
```

| Category | Strength | Weakness |
