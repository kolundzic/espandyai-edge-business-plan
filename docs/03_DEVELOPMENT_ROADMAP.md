# 03 — Development Roadmap

## Phase 0 — Canon Lock

### Goal

Lock the product definition, target audience, architecture, and first prototype scope.

### Deliverables

```text
README.md
Business plan
Architecture document
Hardware spec
Pilot offer
Investor one-pager
Legal positioning
```

### Exit Criteria

The project can be explained clearly to a developer, investor, and pilot customer.

---

## Phase 1 — Proof of Concept

### Goal

Build one functional prototype that proves the device can communicate with Director.

### Required Functions

```text
push-to-talk
microphone capture
Wi-Fi connection
Director API call
voice/text response
speaker output
local SD event log
simple LED or screen status
```

### Estimated Time

```text
2–4 weeks
```

### Estimated Cost

```text
$50–$170 per prototype
```

### Exit Criteria

A short demo video shows the device receiving a voice command, sending it to Director, receiving a response, speaking back, and logging the event.

---

## Phase 2 — Engineering Prototype

### Goal

Build 5–10 cleaner prototypes for demos and early pilot conversations.

### Added Functions

```text
device ID
basic setup flow
status screen
stable enclosure
local logs
basic dashboard view
safe command list
```

### Estimated Time

```text
1–2 months after PoC
```

### Estimated Budget

```text
$900–$3,200
```

### Exit Criteria

The device can be shown to a partner without appearing like a fragile breadboard experiment.

---

## Phase 3 — Dev Kit Batch

### Goal

Produce 20–50 early kits for developers, partners, and pilot users.

### Deliverables

```text
assembled devices
setup guide
firmware image
Director connector
sample workflows
support channel
feedback form
```

### Estimated Budget

```text
$4,800–$14,500 for 50 units
```

### Exit Criteria

At least 20 external users can test the system.

---

## Phase 4 — Business Pilot

### Goal

Deploy 2–5 pilot projects with real customers.

### Target Pilot Types

```text
office node
documentation assistant
workshop/lab node
education node
hotel/reception node
```

### Pilot Pricing

```text
Light pilot: $1,500–$3,000
Business pilot: $5,000–$15,000
Advanced pilot: $15,000–$30,000
```

### Exit Criteria

At least one paid pilot or signed letter of intent.

---

## Phase 5 — Investor Pack

### Goal

Prepare the project for pre-seed fundraising.

### Deliverables

```text
pitch deck
investor one-pager
financial model
demo video
pilot results
waitlist metrics
product roadmap
technical architecture
risk register
```

### Suggested Pre-Seed Ask

```text
Mini round: $50,000–$150,000
Serious pre-seed: $250,000–$500,000
```

---

## Phase 6 — First Commercial Series

### Goal

Move from dev kit to commercial product line.

### Batch Strategy

```text
1 unit: proof of concept
5–10 units: engineering prototype
20–50 units: dev kit batch
100–300 units: pilot commercial batch
500–1,000 units: first commercial series
5,000+ units: scale
```

### Exit Criteria

A repeatable product with support, pricing, documentation, and sales pipeline.
