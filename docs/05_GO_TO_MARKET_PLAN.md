# 05 — Go-To-Market Plan

## 1. Strategy

Do not launch as a mass consumer gadget first.

Launch in this order:

```text
demo
pilot
dev kit
business package
commercial series
platform
```

## 2. First Public Offer

### ESPAndyAI EDGE Dev Kit

Audience:

```text
developers
AI builders
makers
technical partners
early adopters
```

Positioning:

```text
Build your own physical AI node connected to AndyAI Director.
```

Suggested Price:

```text
$199–$349
```

small businesses
