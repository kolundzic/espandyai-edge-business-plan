# Investor Visual System v1.1

## Purpose

This document defines the visual language for investor, partner, and pilot-client communication.

The goal is not decoration.

The goal is business clarity.

Visuals must help investors understand:

```text
what the product is
where the intelligence lives
why the device matters
how the business scales
why trust/governance matters
```

---

## Core visual metaphor

```text
AndyAI Director = brain / command center
ESPAndyAI EDGE Node = body / voice / physical presence
Human founder/operator = final authority
```

Canonical line:

```text
Small Node. Big Brain. Human Command.
```

---

## Visual style

Preferred style:

```text
clean technical editorial
white/light background
black/charcoal text
red/gold accent
simple premium diagrams
isometric device views
clear callout labels
no noisy cyberpunk clutter
```

## Visual rule

Every visual must answer one business question.

If a visual only looks beautiful but does not explain value, it is not ready.
