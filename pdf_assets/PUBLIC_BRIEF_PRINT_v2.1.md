# ESPAndyAI EDGE — Public Brief v2.1

## Give your AI a body.

ESPAndyAI EDGE is a physical AI node concept that connects small ESP32-class devices to AndyAI Director.

The goal is simple:

```text
voice
physical presence
local logging
workflow execution
human approval
business memory
```

This is not a consumer smart speaker.

It is a business-oriented, human-governed AI edge-node system.

---

## What the device does

The node can:

```text
listen through a microphone
speak through a speaker
capture button events
store local logs on SD memory
connect to Director over Wi‑Fi
send workflow events
receive approved actions
```

---

## What Director does

AndyAI Director provides:

```text
memory
policy rules
approval gates
task routing
document/workflow context
dashboard
audit trail
```

---

## Why it matters

AI must move from passive chat into real environments, but it must not lose human control.

ESPAndyAI EDGE is built around that balance:

```text
physical AI interface
business workflow value
human authority
auditable action
```
